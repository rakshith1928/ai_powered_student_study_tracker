# AI-Based Student Study Tracker

A **Streamlit**-based Student Study Tracker with AI-powered insights to help students manage their study schedules, track progress, and improve productivity. This app integrates **Machine Learning models** (Decision Tree, KNN) for personalized recommendations and insights.

---

## Features

- **Login & Authentication**: User login system implemented within Streamlit.  
- **Dashboard**: Visual overview of study progress, completed tasks, and pending backlogs.  
- **Progress Tracker**: Track subject-wise performance and visualize progress over time.  
- **Backlogs Management**: View and manage unfinished or pending tasks/assignments.  
- **AI Insights**: Machine Learning models analyze study data to provide smart recommendations.  
- **Planner Dashboard**: Plan daily/weekly study schedules with AI guidance.  

---

## Tech Stack

- **Frontend & Lightweight Backend**: Streamlit (handles UI, authentication, data operations)  
- **Machine Learning**: Scikit-learn (Decision Tree, KNN)  
- **Data Storage**: CSV/SQLite for tracking progress and tasks  
- **Visualization**: Matplotlib & Seaborn  

---

## Installation

1. **Clone the repository**
````bash
git clone <your-repo-url>
cd <repo-folder>
````

2. **Install dependencies**
````bash
pip install -r requirements.txt
````

3.**Run the App**
````bash
streamlit run ai-studytracker.py
````
